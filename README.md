# ⚡ PC OPTIMIZER PRO v3.0

Программа для увеличения FPS, снижения input lag и устранения фризов/микрофризов в играх на Windows 10/11.

---

## 📦 Содержимое архива

| Файл | Назначение |
|---|---|
| `optimizer.py` | Исходный код программы (Python) |
| `BUILD.bat` | Скрипт сборки .exe одним кликом |
| `requirements.txt` | Список зависимостей Python |
| `icon.ico` | Иконка приложения (опционально, свою можно заменить) |
| `README.md` | Этот файл |

---

## 🚀 Быстрый старт (для продавца / сборки .exe)

1. Установи **Python 3.10+**: https://python.org (отметь галочку *"Add Python to PATH"* при установке)
2. Положи все файлы архива в одну папку
3. Запусти **`BUILD.bat`** двойным кликом
4. Подожди 1-2 минуты — откроется папка `dist/` с готовым **`PC_Optimizer_Pro.exe`**
5. `PC_Optimizer_Pro.exe` — это и есть финальный продукт, который можно продавать / раздавать клиентам. Больше ничего для запуска не требуется (Python на компьютере пользователя не нужен).

> 💡 Совет: если хочешь свою иконку — положи файл `icon.ico` (любой конвертер PNG→ICO в помощь) в ту же папку перед запуском `BUILD.bat`.

---

## 🖥️ Как пользоваться готовым PC_Optimizer_Pro.exe

1. Запусти `PC_Optimizer_Pro.exe` — появится запрос прав администратора, нажми **"Да"**
2. Выбери процессор: **Intel** или **AMD**
3. Выбери видеокарту: **NVIDIA** или **AMD**
4. При желании — сними галочки с ненужных категорий твиков
5. Нажми **"⚡ НАЧАТЬ ОПТИМИЗАЦИЮ"**
6. Дождись завершения (лог покажет все применённые твики)
7. **Перезагрузи ПК** — часть твиков (таймер, C-States, x2APIC) активируется только после перезагрузки

## 🔧 Что оптимизирует программа

### Общие твики (все конфигурации)

**⏱ Таймер / DPC**
- Схема питания «Максимальная производительность»
- Отключение CPU Idle States и Power Throttling
- Отключение динамического тика и синхронизация TSC (ниже DPC latency)
- GPU Hardware Scheduling (HAGS)
- MSI Mode для видеокарты — меньше input lag на прерываниях
- Приоритет CPU/GPU для игр выставлен на «High»

**🌐 Сеть / Input Lag**
- Отключение сетевого троттлинга Windows
- TCP NoDelay / отключение алгоритма Нэгла
- RSS, TCP Chimney Offload, автотюнинг TCP
- DirectX Max Frame Latency = 1
- Отключение ускорения мыши, увеличение очереди клавиатуры
- Отключение Fullscreen Optimizations и Xbox Game Bar/DVR
- Включение Windows Game Mode

**🎮 GPU Мощность**
- Отключение PCI-E Power Management (ASPM) — GPU всегда на полной мощности
- Минимизация визуальных эффектов и анимаций DWM

**🔧 Сервисы**
- Отключение телеметрии Microsoft, SysMain, индексации поиска
- Отключение геолокации, факса, Delivery Optimization, рекламных сервисов
- Отключение автообслуживания Windows и уведомлений

**💾 Память / Диск**
- Фиксированный файл подкачки 8 ГБ (устраняет микрофризы от его расширения)
- Оптимизация NTFS (отключение Last Access Timestamp, 8.3-имена)
- Включение TRIM для SSD
- Очистка временных файлов и кэша миниатюр

### Твики под процессор

**Intel:** отключение Intel PPM, фиксация максимальной частоты, отключение Core Parking, включение x2APIC, фиксация Turbo Boost, ограничение C-States.

**AMD Ryzen:** активация плана питания Ryzen High Performance, отключение Cool'n'Quiet/PPM, отключение Core Parking, TSC Sync Enhanced, приоритет Precision Boost Overdrive, SMT Idle оптимизация.

### Твики под видеокарту

**NVIDIA:** отключение телеметрии и GeForce Overlay, Display Power Saving выкл., PowerMizer на максимум производительности (все GPU-слоты), отключение HDCP, Persistence Mode + отключение auto-boost через nvidia-smi.

**AMD Radeon:** отключение ReLive/Overlay, отключение ULPS (GPU не уходит в глубокий сон между кадрами), включение Anti-Lag, отключение Deep Sleep (SCLK/MCLK) и Compute Preemption.

## ⚠️ Важно

- Программе **обязательно** нужны права администратора — без них большинство твиков не применится
- Рекомендуется создать **точку восстановления Windows** перед первым запуском (Панель управления → Восстановление → Создать точку восстановления)
- Некоторые твики (x2APIC, dynamic tick, C-States) требуют **перезагрузки** для вступления в силу
- Программа не выполняет разгон (overclocking) и не изменяет напряжения — только программные настройки Windows и драйверов
- Протестировано на Windows 10 22H2 и Windows 11 23H2/24H2
